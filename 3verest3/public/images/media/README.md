# Media Images Placeholder Directory

This directory contains placeholder images for the 3verest Media page.

## Required Images

The following placeholder images are referenced in the Media page components:

### Featured Story
- `featured-placeholder.jpg` - Main featured story image (recommended: 1200x800px)

### Media Grid
- `placeholder-1.jpg` - Media card image 1 (recommended: 800x600px for featured, or 600x400px)
- `placeholder-2.jpg` - Media card image 2 (recommended: 600x400px)
- `placeholder-3.jpg` - Media card image 3 (recommended: 600x400px)

## Image Guidelines

- **Format**: JPEG or PNG
- **Aspect Ratio**: 3:2 or 4:3 for standard cards, 2:1 for featured
- **Content**: Should reflect healthcare cloud infrastructure, data centers, medical technology, or related themes
- **Color Palette**: Align with 3verest brand colors (#00AEEF, #32F5C8, dark backgrounds)
