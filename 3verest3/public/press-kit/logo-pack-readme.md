## File Format Guide

### SVG (Scalable Vector Graphics)
- **Use for**: Websites, apps, any digital application requiring scaling
- **Benefits**: Infinitely scalable, small file size, editable
- **Recommended for**: Web development, responsive design

### PNG (Portable Network Graphics)
- **Use for**: Email signatures, social media, presentations
- **Resolutions included**:
  - @1x (standard resolution)
  - @2x (retina displays)
  - @3x (high-DPI screens)
- **Recommended for**: General digital use, PowerPoint, Word documents

### EPS (Encapsulated PostScript)
- **Use for**: Print production, professional design software
- **Color mode**: CMYK
- **Recommended for**: Brochures, business cards, large format printing

### PDF (Portable Document Format)
- **Use for**: Print-ready deliverables, presentations requiring vector graphics
- **Recommended for**: High-quality print materials, archival purposes

## Logo Usage Guidelines

### Minimum Sizes

**Digital**
- Full logo: Minimum 120px width
- Logo mark: Minimum 32px width

**Print**
- Full logo: Minimum 25mm width
- Logo mark: Minimum 8mm width

### Clear Space

Always maintain clear space around the logo equal to the height of the "3" character on all sides. This ensures the logo remains visually distinct and uncluttered.

### Color Specifications

**Primary Logo Color**
- Digital (RGB): 0, 174, 239
- Print (CMYK): 100, 27, 0, 6
- Web (HEX): #00AEEF

**On Dark Backgrounds**
- Use white logo (RGB: 255, 255, 255)
- Or use primary blue (#00AEEF)

**On Light Backgrounds**
- Use dark logo (RGB: 10, 15, 26 / HEX: #0A0F1A)
- Or use primary blue (#00AEEF)

### Background Requirements

**White Logo**
- Requires dark background
- Minimum contrast ratio: 4.5:1
- Recommended backgrounds: #0A0F1A, #0F1C29, or darker

**Dark Logo**
- Requires light background
- Minimum contrast ratio: 4.5:1
- Recommended backgrounds: #FFFFFF, #F0FFFB, or lighter

**Blue Logo**
- Works on both light and dark backgrounds
- Ensure sufficient contrast
- Best on white or very dark backgrounds

## Correct Usage

✅ **Do**:
- Maintain original proportions (no stretching)
- Use appropriate color version for background
- Maintain clear space requirements
- Use high-resolution files for print
- Use vector formats (SVG, EPS) when possible

❌ **Don't**:
- Stretch, skew, or distort the logo
- Add effects, shadows, or gradients
- Change the logo colors (use provided versions)
- Place on busy backgrounds that reduce legibility
- Rotate the logo at angles
- Separate the "3" from "VEREST"
- Recreate or modify the logo

## Usage Scenarios

### Website Header
**Use**: 3verest-logo-primary-white.svg (on dark nav) or 3verest-logo-primary-dark.svg (on light nav)
**Size**: 120-180px width

### Email Signature
**Use**: 3verest-logo-primary-dark.png @2x
**Size**: 150px width

### Social Media Profile
**Use**: 3verest-mark-blue.png @3x
**Size**: Platform-specific (usually 400x400px or larger)

### PowerPoint Presentation
**Use**: 3verest-logo-primary-white.png @2x (dark slides) or 3verest-logo-primary-dark.png @2x (light slides)
**Size**: 2-3 inches width

### Print Business Card
**Use**: 3verest-logo-cmyk.eps or 3verest-logo-cmyk.pdf
**Size**: 20-25mm width minimum

### Large Format Print
**Use**: 3verest-logo-cmyk.eps or SVG
**Size**: As needed (vector files scale infinitely)

### Favicon / App Icon
**Use**: 3verest-mark-blue.png @3x
**Size**: Create at required dimensions (16x16, 32x32, 192x192, etc.)

## Typography in Logo

**Typeface**: Custom modified Montserrat
**Characteristics**:
- Reversed "E" (Ǝ) representing "3"
- "VEREST" in uppercase
- Specific kerning and letter spacing
- Weight: Semibold (600)

**Note**: Do not attempt to recreate the logo using standard fonts. Always use provided logo files.
